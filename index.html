About This Project

This project is a Spotify Clone built using HTML and CSS. The project focuses on building a responsive and interactive layout without using JavaScript. It includes a sidebar navigation, music cards, trending sections, and a functional music player layout.

Key Features:

  • 🎧 Sidebar navigation for Home, Search, and Library
  • 💿 Music cards showing trending songs, albums, and featured charts
  • 🎵 Fixed music player at the bottom with playback controls
  • 💡 Hover effects and smooth styling inspired by Spotifyâ€™s theme

Technologies Used:

HTML, CSS, Font Awesome, Visual Studio Code
